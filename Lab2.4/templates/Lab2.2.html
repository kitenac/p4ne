Создайте веб-сервер, который делает следующее:
a) При обращении по “/” — выдаёт краткую справку об использовании
b) При обращении по “/configs” — выдаёт сведения об именах всех хостов, для которых есть кофигурационные файлы (см. работу 1.6)
